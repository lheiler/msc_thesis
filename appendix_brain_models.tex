\section{Computational Brain Model Fitting Results}
\label{app:brain_models}

This appendix presents detailed fitting results for all computational brain models tested on real EEG data. Each model was evaluated on 5 randomly selected EEG samples, with comprehensive comparison plots showing model PSD fit quality, residuals, and frequency band analysis.

\subsection{Model Overview}

The following computational brain models were evaluated:
\begin{itemize}
    \item \textbf{CTM-CMA-AVG}: Cortico-Thalamic Model with CMA-ES optimization (channel-averaged)
    \item \textbf{CTM-NN-AVG}: Cortico-Thalamic Model with neural network inference (channel-averaged)
    \item \textbf{CTM-NN-PC}: Cortico-Thalamic Model with neural network inference (per-channel)
    \item \textbf{JR-AVG}: Jansen-Rit Model (channel-averaged)
    \item \textbf{JR-PC}: Jansen-Rit Model (per-channel)
    \item \textbf{Hopf-AVG}: Hopf Oscillator Model (channel-averaged)
    \item \textbf{Hopf-PC}: Hopf Oscillator Model (per-channel)
    \item \textbf{Wong-Wang-AVG}: Wong-Wang Model (channel-averaged)
\end{itemize}

\subsection{Discussion}

The comprehensive comparison reveals several key findings:

\begin{itemize}
    \item \textbf{Speed vs. Quality Trade-off}: CTM-NN models achieve comparable fit quality to CMA-ES optimization while being orders of magnitude faster.
    
    \item \textbf{Model Complexity}: Per-channel models generally require more computation time but may capture spatial heterogeneity better than channel-averaged approaches.
    
    \item \textbf{Neural Network Efficiency}: The amortized neural network approach demonstrates potential for real-time brain state estimation.
    
    \item \textbf{Model-Specific Performance}: Different computational brain models show varying success in capturing EEG spectral characteristics, with CTM variants showing particularly strong performance.
\end{itemize}
